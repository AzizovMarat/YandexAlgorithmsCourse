<p>В данном списке из n ≤ 10<sup>5</sup> целых чисел найдите три числа,произведение которых максимально.
<br>Решение должно иметь сложность O(n), где n - размер списка.</p>
<p>Выведите три искомых числа в любом порядке.</p>
