<p>Даны два списка чисел, которые могут содержать до 10000 чисел каждый. Выведите все числа, которые входят как в первый, так и во второй список в порядке возрастания. Примечание. И даже эту задачу на Питоне можно решить в одну строчку.</p>

<h2>Формат ввода</h2>
<p>Вводятся два списка целых чисел. Все числа каждого списка находятся на отдельной строке.</p>

<h2>Формат вывода</h2>
<p>Выведите ответ на задачу.</p>
